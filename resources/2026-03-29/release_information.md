# AI開発ツール リリース情報（2026-03-22〜2026-03-29）

## 1. Gemini CLI（v0.35.0 〜 v0.35.3）

**リリース日**: 2026-03-24（v0.35.0）、2026-03-26（v0.35.1/v0.35.2）、2026-03-28（v0.35.3）
**リポジトリ**: https://github.com/google-gemini/gemini-cli
**リリースページ**: https://github.com/google-gemini/gemini-cli/releases/tag/v0.35.3

今週は v0.35.0 から v0.35.3 まで4つの安定版リリースが公開された。特に v0.35.0 は大規模なフィーチャーリリースで、多くの新機能が追加されている。

### 主な新機能（v0.35.0）

- **カスタマイズ可能なキーボードショートカット**: ユーザー独自のキーバインディング設定が可能になった。`-` プレフィックスによるキーバインディングの無効化もサポート。Kittyプロトコル拡張キーにも対応。
- **Vim モード強化**: `X`、`~`、`r`、`f/F/t/T`、`df/dt`、`y/p/P`（ヤンク＆ペースト）など、多くのVimモーションが追加された。
- **サブエージェント機能の正式有効化**: サブエージェントが有効化され、ツール分離の基盤、手動削除の修正、コンテキスト伝播の改善が含まれる。
- **サンドボックス機能の実装**: `SandboxManager` インターフェースとスキーマが実装され、すべてのプロセス生成ツールにサンドボックスが統合された。Linux向けにbubblewrapとseccompによるサンドボックスも追加。
- **JIT（Just-In-Time）コンテキストローディング**: ファイルシステムツールを通じたJITコンテキスト発見が実装され、デフォルトで有効化された。
- **モデル駆動の並列ツールスケジューラ**: モデルが判断する並列ツール実行スケジューラが実装され、安全なツールはエージェントがビジー中でも並列実行可能になった。
- **`--admin-policy` フラグ**: 管理者ポリシーを補足的に追加するためのフラグが導入された。
- **カスタムベースURL対応**: 環境変数によるカスタムベースURLの設定がサポートされた。
- **A2Aネイティブ gRPC サポート**: Agent-to-Agent通信でネイティブgRPCとプロトコルルーティングが有効化された。
- **ブラウザエージェントのドメイン制限**: ブラウザエージェントに対して許可ドメインの制限を設定可能になった。
- **拡張機能の暗号整合性検証**: 拡張機能更新時の暗号的整合性検証が実装された。
- **`disableAlwaysAllow` 設定**: 自動承認を無効にするセキュリティ設定が追加された。
- **Webfetch ツールの改善**: Stage 1/Stage 2 のセキュリティおよび一貫性改善が実施された。
- **Topic-Action-Summary モデル**: 冗長性を削減するための新しいプロンプト最適化モデルが実装された。

### バグ修正・改善

- APIエラー時の自動リトライ（最大3回）
- CJK入力とUnicodeスカラー値のサポート改善
- コピーモードでのスクロールキー対応
- セッション再開時の動的ツール説明の保持
- ポリシーエンジンへのサブエージェントコンテキスト伝播の修正
- コード分割と遅延UIローディングによるパフォーマンス最適化

### パッチリリース（v0.35.1 〜 v0.35.3）

- **v0.35.1**: マイナー修正
- **v0.35.2**: マイナー修正
- **v0.35.3**: サンドボックス設定でのネットワークアクセス有効化（cherry-pick）

## 3. Cursor（Self-hosted Cloud Agents）

**リリース日**: 2026-03-25
**ウェブサイト**: https://cursor.sh
**Changelog**: https://cursor.com/changelog

### Self-hosted Cloud Agents

Cursorにセルフホスト型クラウドエージェントが導入された。主な特徴：

- **コードとツール実行を自社ネットワーク内に完全保持**: コードベース、ビルド出力、シークレットが内部インフラストラクチャ上で管理される。
- **ツールコールのローカル処理**: エージェントのツールコールがローカルで実行される。
- **Cursorホスト版と同等の機能**: 分離された仮想マシンとプラグインサポートを含む完全な機能パリティを提供。

企業のセキュリティ要件を満たしながら、Cursorのクラウドエージェント機能を利用できるようになった。

### チャット体験の改善

- **統合カスタマイズインターフェース**: カスタム命令、プロンプトファイル、エージェント、スキルを統合UIで一元管理。構文ハイライトとバリデーション対応。
- **推論深度の設定可能なコントロール**: Claude Sonnet 4.6やGPT-5.4など推論対応モデルで、モデルピッカーから直接推論深度サブメニューにアクセス可能。
- **画像プレビュー強化**: チャット添付画像をフル機能の画像ビューアで閲覧可能（ナビゲーション、ズーム、パン対応）。

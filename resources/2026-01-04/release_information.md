## 2. GitHub Copilot CLI v0.0.374

**リリース日:** 2026-01-02
**リポジトリ:** https://github.com/github/copilot-cli
**リリースページ:** https://github.com/github/copilot-cli/releases/tag/v0.0.374

### 主な変更点

v0.0.374では複数の重要な新機能と改善が追加されました。

#### 新機能

1. **自動コンパクション機能**
   - トークン制限の95%に達すると自動的にコンパクションを実行
   - `/compact` コマンドで手動実行も可能
   - 長時間のセッションでもコンテキストを効率的に管理

2. **ビルトインサブエージェント**
   - タスクの探索と管理のための組み込みサブエージェントを搭載
   - 複雑なタスクを並列で処理可能に

3. **`web_fetch` ツール**
   - Webコンテンツを取得するためのビルトインツールを追加
   - 外部リソースの参照が容易に

#### 改善

- **MCPサーバータイプ:** ヘルプテキストで正しいオプションを表示
- **モデルピッカー:** モデルが利用不可の場合、設定リンク付きのより明確なメッセージを表示

### 使用例

```bash
# コンテキストの手動コンパクション
/compact

# 自動コンパクションは95%のトークン制限で自動実行
```

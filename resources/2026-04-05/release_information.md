# リリース情報まとめ（2026-03-29 〜 2026-04-05）

## 1. Gemini CLI v0.36.0（2026-04-01）

**リポジトリ**: https://github.com/google-gemini/gemini-cli
**リリース**: https://github.com/google-gemini/gemini-cli/releases/tag/v0.36.0

### 主な新機能

- **サブエージェントのローカル実行とツール分離**: マルチレジストリアーキテクチャを導入し、サブエージェントごとにツールのフィルタリングが可能に。ツール拒否時にはコンテキストに応じたフィードバックを返すレジリエントな仕組みも追加
- **Git Worktreeサポート**: 並列セッション用の分離されたGit worktreeを作成可能に。テレメトリでworktreeセッションかどうかも追跡
- **macOS/Windowsネイティブサンドボックス**: macOSではSeatbeltを利用した厳格なサンドボックス、Windowsではネイティブサンドボックスをそれぞれ実装。動的なサンドボックス拡張とworktreeサポートも追加
- **実験的メモリマネージャエージェント**: 従来の`save_memory`ツールを置き換える新しいメモリマネージャエージェントを追加
- **タスクトラッカープロトコル統合**: コアシステムプロンプトにタスクトラッカーを統合し、タスクやToDo項目に「blocked」ステータスを追加
- **A2Aエージェント認識コマンド**: エージェント認識コマンドとレジストリ発見機能を強化
- **プランモードの非インタラクティブ対応**: 非インタラクティブモードでもプランモードが利用可能に
- **管理者によるMCPサーバーの強制インストール**: 管理者がMCPサーバーのインストールを強制できる機能を追加
- **Composerレイアウトの新UX**: リフレッシュされたComposerレイアウトのUXを実装
- **起動中の即座の入力受付**: 起動完了前からユーザー入力を受け付け可能に
- **ブラウザエージェントのセキュリティ強化**: セキュリティプロンプト、センシティブアクション制御、読み取り専用モードでのノイズ削減を追加

### その他の改善

- JITコンテキストのgitルートでの上方トラバーサル制限
- ポリシーエンジンのtoolName指定の強制化
- VSCode ワークスペースフォルダーの自動コンテキスト追加
- `--version`起動時間の最適化
- 起動クリーンアップタスクの並列化とバックグラウンド実行

## 3. Cursor 3.0（2026-04-02）

**ウェブサイト**: https://cursor.sh
**Changelog**: https://cursor.com/changelog

### 主な新機能

- **Agents Window**: 新しいインターフェースにより、ローカル、worktree、クラウド、リモートSSHなど複数の環境・リポジトリで多数のエージェントを並列実行可能。`Cmd+Shift+P → Agents Window`で切り替え
- **Design Mode**: Agents Windowに統合されたブラウザ上のUI要素アノテーション機能
  - `⌘+Shift+D`でDesign Modeの切替
  - `Shift+ドラッグ`でエリア選択
  - `⌘+L`で要素をチャットに追加
  - `⌥+クリック`で要素を入力に追加
- **Agent Tabs**: サイドバイサイドやグリッドレイアウトで複数チャットを同時表示
- **`/worktree`コマンド**: 分離されたgit worktreeを作成
- **`/best-of-n`コマンド**: タスクを複数モデルで並列実行し、結果を比較
- **`Await`ツール**: エージェントがバックグラウンドプロセスを管理するための新ツール

### エンタープライズ・チーム向け

- 監査ログにディレクトリグループ名を追加
- クラウドエージェントのシークレット管理を管理者に限定するチームレベル設定
- 「Made with Cursor」コード帰属表示の企業全体での無効化

### パフォーマンス改善

- 大規模ファイルの差分レンダリングの高速化とメモリ使用量削減
- Explorerサブエージェントのキャッシング改善
- MCPアプリがリッチな出力のための構造化コンテンツに対応

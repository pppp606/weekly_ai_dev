## Claude Code (Anthropic)

**バージョン**: 2.0.68 / 2.0.67
**更新日**: 2025-12-13
**リポジトリ**: https://github.com/anthropics/claude-code
**CHANGELOG**: https://github.com/anthropics/claude-code/blob/main/CHANGELOG.md

### 主な変更点

#### v2.0.68
- **IME対応改善**: 中国語・日本語・韓国語などのIME（入力メソッドエディタ）サポートを修正、カーソル位置に変換ウィンドウが正しく表示されるように
- **MCPツールの修正**: 許可されていないMCPツールがモデルに表示されないよう修正
- **サブエージェント中のステアリングメッセージ**: サブエージェント動作中にステアリングメッセージが失われる問題を修正
- **CJKテキストナビゲーション**: Option+矢印キーでのワードナビゲーションがCJKテキストシーケンス全体を単一の単語として扱う問題を修正
- **プランモード終了UX改善**: 空または欠落したプランで終了時に、エラーの代わりに簡略化されたyes/noダイアログを表示
- **エンタープライズ管理設定サポート**: 企業向けマネージド設定のサポートを追加

#### v2.0.67
- **Opus 4.5でThinkingモードがデフォルト有効**: Opus 4.5使用時にthinkingモードがデフォルトで有効に
- **Thinkingモード設定の移動**: `/config`に移動
- **権限検索機能**: `/permissions`コマンドに検索機能を追加、`/`キーでツール名によるフィルタリングが可能
- **非ラテン文字対応**: キリル文字、ギリシャ語、アラビア語、ヘブライ語、タイ語、中国語でのワード削除・ナビゲーションを修正

### 開発者への影響
- 日本語入力時のIME体験が大幅に改善
- エンタープライズ環境での導入がより容易に
- Opus 4.5利用時は自動的にthinkingモードが有効になり、より深い推論が可能

## Cursor

**バージョン**: 2.2
**リリース日**: 2025-12-10
**公式サイト**: https://cursor.sh
**CHANGELOG**: https://cursor.sh/changelog

### 主な新機能

#### Debug Mode（デバッグモード）
- **概要**: アプリにランタイムログを計装し、根本原因を特定
- **使用方法**: バグ発見時にデバッグモードを起動すると、自動的にログを挿入して問題の原因を追跡

#### Browser Layout & Style Editor
- **概要**: デザインとコーディングを同時に実行できるブラウザレイアウト機能
- **特徴**: UIの変更をリアルタイムでプレビューしながらコード編集が可能

#### Plan Mode強化
- **Mermaidダイアグラム対応**: プランモードでインラインMermaidダイアグラムの自動生成をサポート
- **視覚的な設計**: 複雑な実装計画を図解で表現

#### Multi-Agent Judging
- **概要**: 自動評価機能により、最適なソリューションを推奨
- **用途**: 複数の実装アプローチから最適解を選択

#### その他の改善
- **Pinned Chats**: エージェントサイドバーにチャットのピン留め機能
- ブラウザ保護コントロールの追加
- プランファイルの保存機能
- 計10項目の追加改善

## VS Code

**バージョン**: 1.107.0 (November 2025)
**リリース日**: 2025-12-10
**リポジトリ**: https://github.com/microsoft/vscode
**リリースURL**: https://github.com/microsoft/vscode/releases/tag/1.107.0

### AI関連の主な新機能

#### Agent HQ
- **概要**: GitHub Copilotと他のコーディングエージェントを併用して複数タスクを完了
- **特徴**: バックグラウンド、クラウド、ローカルエージェントの強みを活かして適材適所で利用
- **使用シーン**: 複数の開発タスクを並行して処理

#### Background Agents（バックグラウンドエージェント）
- **概要**: 独立した隔離ワークスペースでバックグラウンド動作
- **管理**: VS Codeが自動管理

#### Custom Agents for Organizations
- **概要**: 組織レベルでカスタムエージェントを共有可能に
- **用途**: チーム全体で統一されたエージェント設定を利用

### 開発者への影響
- 複数エージェントの同時利用により、開発効率が大幅に向上
- バックグラウンドエージェントにより、作業中断なく並行タスク処理が可能
- 企業・チームでのAI開発ツール標準化が容易に

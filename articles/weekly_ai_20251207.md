## 🚀 リリース情報

### Claude Code v2.0.60

**リリース日**: 2025-12-06

バックグラウンドエージェントサポートが追加され、エージェントをバックグラウンドで実行しながら作業を継続できるようになりました。複数のタスクを並行して処理できるため、開発効率の向上が期待できます。

その他の新機能:
- `--disable-slash-commands` CLIフラグでスラッシュコマンドを無効化
- `/mcp enable/disable [server-name]` でMCPサーバーを素早く切り替え
- VSCodeで複数のターミナルクライアントが同時にIDEサーバーに接続可能

過去1週間では、v2.0.58でProユーザーがOpus 4.5にアクセス可能になりました。

https://github.com/anthropics/claude-code

### Cline v3.40.1 / v3.40.0

**リリース日**: 2025-12-04〜06

- **Gemini 3 Pro モデル対応**: デフォルトの思考レベルを設定可能に
- **Atomic File Write**: ファイル書き込みの原子性を確保
- **タスク履歴の自動リカバリ**: 破損したタスク履歴状態を自動回復

https://github.com/cline/cline

### GitHub Copilot CLI v0.0.367

**リリース日**: 2025-12-05

GPT-5.1-Codex-Maxに対応し、コマンドラインでの開発支援が強化されました。

https://github.com/github/copilot-cli

### Kiro v0.7

**リリース日**: 2025-12-03

新機能「Powers」を導入。会話コンテキストに基づいてMCPツールをオンデマンドでアクティブ化する動的ローディング機能です。コンテキスト容量80%で自動要約がトリガーされる会話要約機能も追加されました。

https://kiro.dev/changelog/

## 📈 注目のAI開発リポジトリ

### microsoft/magentic-ui

https://github.com/microsoft/magentic-ui

Microsoftが公開した「人間中心」のWebエージェント研究プロトタイプ。従来のAIエージェントがブラックボックス的に動作するのに対し、行動計画を事前に提示してユーザーが実行をガイドできる設計になっています。

主な機能:
- **Co-Planning**: エージェントと一緒にステップバイステップの実行計画を策定
- **Co-Tasking**: 実行中にチャットやブラウザ直接操作で介入可能
- **Action Guards**: 決済など機密性の高い操作には明示的な承認を要求

AutoGenベースで構築されており、OpenAI、Azure、Ollamaなど複数のLLMプロバイダに対応。MCPサーバー統合もサポートしています。

### aliasrobotics/cai

https://github.com/aliasrobotics/cai

セキュリティ専門家向けのAI搭載サイバーセキュリティフレームワーク。攻撃側と防御側の両方のセキュリティ自動化をAIエージェントで実現します。300以上のAIモデルに対応しており、HackTheBox CTFやMistral AIコンペで上位入賞した戦略を組み込んでいます。

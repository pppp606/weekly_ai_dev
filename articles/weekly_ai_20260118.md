## 🚀 リリース情報

### Claude Code (v2.1.10 - v2.1.12)
**リポジトリ**: https://github.com/anthropics/claude-code

- **Setupフックイベント**: `--init`、`--init-only`、`--maintenance` CLIフラグで起動可能な新しいフックイベント。プロジェクト固有の初期化やメンテナンスタスクを自動化
- **OAuth URLコピーショートカット**: ログイン時にブラウザが自動で開かない場合、'c'キーでOAuth URLをコピー可能に
- **MCP接続安定性向上**: HTTP/SSEトランスポートでの過剰な接続リクエストを修正し、長時間セッションでの信頼性が改善
- auto:N構文によりMCP tool search自動有効化のしきい値をコンテキストウィンドウのパーセンテージで設定可能

### GitHub Copilot CLI (v0.0.384)
**リポジトリ**: https://github.com/github/copilot-cli

- **バックグラウンド委譲ショートカット**: `&`プレフィックスでプロンプトをバックグラウンドで実行可能
- **メモリ機能**: リポジトリのメモリをプロンプトに注入し、セッション間でファクトを記憶するためのメモリストレージツールを追加
- **拡張思考モード**: Anthropic Claudeモデルで拡張思考（extended thinking）を有効化
- **GPTモデルの推論努力設定**: ユーザーがgptモデルの推論努力（reasoning effort）を設定可能に

### Cline (v3.49.1 - v3.51.0)
**リポジトリ**: https://github.com/cline/cline

- **gpt-5.2-codex モデルサポート**: OpenAI gpt-5.2-codexモデルをモデルピッカーに追加
- **create-pull-request スキル**: 新規追加
- **Responses API**: サポートされるすべてのモデルで段階的に導入

### OpenAI Codex (v0.87.0)
**リポジトリ**: https://github.com/openai/codex

- **コラボレーション待機**: 1回のリクエストで複数のIDをブロック待機可能に、マルチスレッド調整を簡素化
- **ユーザーシェルコマンド**: ユーザースナップショット下で実行され、エイリアスやシェル設定が反映される
- **TUI承認リクエスト表示**: スポーン/非購読スレッドからの承認リクエストをTUIで表示

### Google Gemini CLI (v0.24.0)

- **セキュリティ強化**: フォルダー信頼サポート、シェルセキュリティポリシー統一、デフォルトで非信頼に設定

### Cursor (CLI Jan 16, 2026)

## 📈 注目のAI開発リポジトリ

### UI-TARS-desktop

https://github.com/bytedance/UI-TARS-desktop

ByteDanceが開発した「画面を見て操作する」GUIエージェント。Vision-Language Model（VLM）を使ってスクリーンショットを解析し、マウスクリックやキーボード入力を実行する。従来のDOMセレクタ依存のツールとは異なり、人間と同じように「視覚ベース」で操作するため、UIの見た目が変わらない限り動作し続ける。独自開発のUI-TARS VLMモデルにより、WebArenaやOSWorldベンチマークで高精度を達成。24,000以上のスターを獲得して急上昇中。

### ralph-claude-code

https://github.com/frankbria/ralph-claude-code

Claude Codeに自律開発ループを実装するためのフレームワーク。PROMPT.mdにプロジェクト要件を書くと、完了するまでClaude Codeが繰り返し実行を続ける。二重ゲート方式による終了検出、レート制限、サーキットブレーカーなどの安全機構を搭載。tmuxモニタリングでリアルタイムダッシュボードによる状態監視も可能。テストカバレッジは100%を達成し、v1.0のリリースも間近。
